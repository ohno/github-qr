GitHubQR
This is an unofficial QR code generator for GitHub profiles and projects. You must follow GitHub's guidelines to use. Icons of QR code, menu and link are provided by Google Fonts. The background is based on Animated Backgrounds. This project depends on QR Code Styling by Denys Kozak. More options are available here. *SVG not work on PowerPoint.
